# count 命令 | 豆包

### v1

```bash
给我写一个名为count的shell，比如如果我输入`ll | count`的时候，可以统计一共多少行输出，再给我一个参数，如果不输入这个参数的时候，就直接输出统计信息，如果输入这个参数的时候，就输出统计信息加ll输出的原生信息（统计信息在最下面即可），然后如果我输入这个参数是大写的，那么就统计信息在最上面，然后也要有个参数可以统计多少列，比如我输入ls | count 列参数 的时候，就输出列信息，行信息与列信息可以组合使用，以上是使用管道符做统计信息，然后还要可以输出某个文件夹下面有多少文件和文件夹，比如我输入count /home，那你要输出/home下面有多少文件夹和文件，再来个文件参数，比如我输入count 文件参数 /home就要统计有多少文件（不包含文件夹），同样的还得有个文件夹参数，只统计文件夹，默认统计全部即可，然后还要有个参数，用来过滤掉隐藏文件夹，比如count 参数 /home/jim/ 这时候.ssh文件夹不用统计进去，然后参数和文件夹路径可以调换，参数之间都能组合使用，比如count -a -b 和count -ab的效果要是一样的，然后还要支持通配符，比如我count /home/jim/*.py，就要统计该路径下有多少以.py结尾的文件
```
